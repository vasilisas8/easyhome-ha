# EasyHome

Интеграция Home Assistant для контроллеров EasyHome по Modbus TCP.

## Возможности

- Подключение контроллера по Modbus TCP
- Добавление устройств
- Удаление устройств

## Установка

HA находится по адресу: http://homeassistant.local:8123

### Установка HACS (магазин кастомных интеграций)

1. Откройте следующую ссылку для установки HACS: https://my.home-assistant.io/redirect/supervisor_addon/?addon=cb646a50_get&repository_url=https%3A%2F%2Fgithub.com%2Fhacs%2Faddons
2. Запустите HACS
3. Перейдите в журнал установки HACS ("Журнал" на панели сверху) и дождитесь окончания загрузки
4. Перезапустите HA (Настройки - Инструменты разработчика - Перезапуск HA)

### Настройка HACS

1. Перейдите в Настройки - Устройства и службы - Добавить интеграцию. Очистите кэш браузера (иначе HACS может не появиться в списке интеграций). Введите "HACS" в поиске и выберите интеграцию HACS.
2. Примите условия и нажмите Отправить. Начнётся процесс аутентификации через GitHub:
    - Скопируйте предоставленный код устройства и перейдите по ссылке https://github.com/login/device.
    - Войдите в свой аккаунт GitHub (если не вошли).
    - Вставьте код устройства и подтвердите авторизацию.
    - После подтверждения закройте вкладку и вернитесь в Home Assistant.
3. В левом меню HA должен появиться пункт "HACS"

### Установка кастомной интеграции EasyHome через HACS

1. Откройте HACS в левом меню.
2. Откройте выпадающее меню в правом верхнем углу экрана - Пользовательские репозитории.
3. Вставьте следующую ссылку репозитория: https://github.com/vasilisas8/easyhome-ha
Тип - интеграция.
4. В поиске наберите EasyHome. В списке должна отобразиться интеграция. Перейдите в нее, нажмите "Сккачать".
5. Перейдите в настройки. Сверху HA сам предложит перезапуститься - нажмите "Подтвердить".
6. Перейдите в Настройки - Устройства и службы - Добавить интеграцию. Введите EasyHome - интеграция должна появиться в списке доступных.

## Настройка

1. При установке интеграции, она сразу предложит ввести IP-адрес контроллера и порт Modbus TCP.
2. После подключения нажать "Настройки" - добавить новые устройства.
3. **!! Для удаления устройства перейти в настройки контроллера - удалить устройство - выбрать нужное из списка. Только после этого удалить его в списке устройств под контроллером. !!**

## Поддерживаемые устройства

- Датчики температуры
- Датчики Влажности
- Датчик Освещенности
- Лампы
- Диммируемые лампы
- Выключатели
